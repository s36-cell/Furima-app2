# FURIMA アプリケーション
Laravel を使用したフリマアプリケーションです。  ユーザーは会員登録後、商品閲覧・いいね・コメント・購入が可能です。
---
## 主な機能
### 商品一覧
- 未ログインでも閲覧可能
- SOLD 表示対応
- 検索機能あり
- いいね数 / コメント数 表示
### 商品詳細
- 画像・説明・カテゴリ・価格・状態表示
- SOLD バッジ表示- いいね機能
- コメント機能
### いいね機能
- ログインユーザーのみ利用可
- ハートアイコン切替（未いいね / いいね済み）
- いいね数リアルタイム表示
### コメント機能
- ログインユーザーのみ利用可
- コメント数表示
### 購入機能
- ログイン必須
- 購入確認ページ
- 住所表示
- 支払い方法選択（選択内容が右側にリアルタイム反映）
- 購入完了で商品が SOLD 状態に変更
### 住所変更機能
- カード UI で編集可能
- 更新後、購入画面へ戻る
- バリデーション対応
---
## 認証機能
- Laravel Fortify 使用
- メール認証必須
- 未認証状態 → 認証メール確認画面へリダイレクト
- メールは *開発環境では log 出力*
